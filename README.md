# photonic_compiler

- **Json Parse:** <span style="color:red">IN PROGRESS</span>.

- **ONNX-ResNet:**

- **Pytorch-LeNet:**

- **Relay_LeNet:**
<span style="color:red">IN PROGRESS</span>.

- **tensor_expression_totorial:**
understanding TVm's Tensor Expression language to find uses in code generation
